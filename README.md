# Cpp_OpenCV-CarNumber-Find
C++_OpenCV+CarNumber Find

- 간단하게 OpenCV 공부 후에 라이브러리들을 응용하여보기 위하여 만들어본 프로젝트 입니다.
- 사진내에서 번호판의 위치를 찾아내는 프로그램입니다.
- 단일의 번호판만 가능합니다.
- 특정 조건의 번호판은 오차범위가 매우 큽니다.
